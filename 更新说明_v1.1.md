## 1. 本版要解决什么

1. **合法全排除日**（全部项目被硬规则 reject、0 个推荐/待复核）时，Phase 3/4 生成 Excel 失败。  
2. 工作区/新 Skill **目录与记录习惯**不清晰，根目录易脏、运行账与维护账混写、多副本改一处漏同步。  
3. 需要可分发的 **完整 Skill** 与 **薄封装入口** 两套正式包。

| 文件 | 变更 |

## 5. 安装注意

1. 解压/安装后跑 `scripts/verify_release_manifest_20260623_v01.py --root <skill根>`，应为 `ok: true`。  
2. 用 `prepare_runtime` 建**独立**运行目录；产物禁止写回 Skill 安装目录。  
3. 敏感配置（账号、Webhook、Token）仍在 Skill **外**。  
4. 钉盘/钉钉依赖本机 `dws` 等环境；包内不含密钥与 Chrome。  
5. 若从旧 Runtime 升级：请把本版 `scripts` 同步进 Runtime 的脚本副本，或重新 prepare；**不要**只改一份副本却不更新权威安装。
